# 以'#'开始的行，被视为注释.                                                                                                                          

# 忽略掉所有文件名是 foo.txt的文件.

foo.txt

# 忽略所有生成的 html文件,

*.html

# foo.html是手工维护的，所以例外.

!foo.html

# 忽略所有.o和 .a文件.

*.[oa]


配置语法：
以斜杠“/”开头表示目录；
以星号“*”通配多个字符；
以问号“?”通配单个字符
以方括号“[]”包含单个字符的匹配列表；
以叹号“!”表示不忽略(跟踪)匹配到的文件或目录；
# /mtk/ 过滤整个文件夹  /mtk/do.c 过滤某个具体文件夹

# !/mtk/one.txt 不要过滤文件夹中的one.txt文件

#过滤整个SourceInsight工程文件夹
/SourceInsight/
